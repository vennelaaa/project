Projects

Claims management systems

Designed and developed a comprehensive claims management system for the insurance industry, streamlining the process of managing claims from submission to settlements.

4 Microservices are developed:

  • AUTHORIZATION MICROSERVICE
  • MEMBER MICROSERVICE
  • CLAIMS MICROSERVICE
  • POLICY MICROSERVICE

The User can insert the Claim details like:

  • SUBMIT CLAIM: 1. Member ID 2. Hospital Name 3. Claim Amount 4. Benefit Id 5. Policy Name

A Claim ID will be generated corresponding to the submitted claim.

CHECK STATUS: The User can check the status of a particular claim with the given claim ID. For instance, Claim is pending as more documents are needed for verification.

VIEW BILLS: The User can see if he/she has any pending bills along with the last premium paid date and late charges.
